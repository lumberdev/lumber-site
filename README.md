# Lumber Website

## Installation

    $ npm install
    
or 
    
    $ yarn install

## Usage

Run the development setup (localhost:3000/):

    $ shadow-cljs watch app
    
Build for production:

    $ shadow-cljs release app

Run the project production server:

    $ clj -m lumber.serve
